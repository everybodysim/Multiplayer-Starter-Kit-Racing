# Starter Kit Racing

A JavaScript/Three.js port of [Kenney's Starter Kit Racing](https://github.com/KenneyNL/Starter-Kit-Racing).

[Live multiplayer demo](https://endel.github.io/Multiplayer-Starter-Kit-Racing/) (server may be shut down after a few weeks)

[Original single-player demo](https://mrdoob.github.io/Starter-Kit-Racing/)
